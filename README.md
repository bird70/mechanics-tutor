# Mechanics Tutor

Interactive NZ NCEA Level 3 Physics – Mechanics learning app.

## Features
- Structured lesson plans (Beginner / Intermediate / Advanced)
- Interactive physics simulations (SUVAT solver, projectile, circular motion, SHM)
- NCEA-style practice exam papers with instant feedback
- Progress persistence via localStorage

## Development
```
npm install
npm run dev
```

## Build
```
npm run build
```

## Tests
```
npm test
```
